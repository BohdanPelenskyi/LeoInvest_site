Leo Invest Site v2

Що всередині:
- index.html — головна сторінка
- invest.html — інвестиційна пропозиція + калькулятор дохідності
- faq.html — FAQ + контакти/форма заявки
- assets/css/style.css — весь дизайн
- assets/js/main.js — мобільне меню, FAQ, форма mailto, анімації
- assets/js/calculator.js — калькулятор дохідності
- assets/img/ — офіційне лого, favicon, hero-зображення
- robots.txt, sitemap.xml, .htaccess

Як завантажити на хостинг:
1. Розпакувати ZIP.
2. Вміст папки leo-invest-site-v2 завантажити у корінь сайту / public_html.
3. Перевірити, що index.html знаходиться саме у корені сайту.
4. Якщо сайт відкривається по домену leoinvest.com.ua — нічого більше змінювати не треба.

Примітка:
Форма заявки працює без сервера: вона формує email через mailto на office@leoinvest.com.ua.
Калькулятор працює повністю на стороні браузера.
